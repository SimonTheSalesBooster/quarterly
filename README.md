# 🐬 Quarterly Review: The Interview

> 20 questions. Three acts. One quarter compressed into clarity.

## The core idea

Most reviews are worksheets you fill out alone, resent, and never look at again. This one is an interview. Someone asks you sharp questions, one at a time. You think out loud. At the end, you get a one-page summary of your quarter and one commitment for the next one.

Three acts, same structure every time:

1. **Appreciate the Quarter** — What worked, what you learned, what made you happy as a dog, what made you fast as a cheetah
2. **Read the Present** — What hurts, what you want, what you're tolerating, what decision you've been avoiding
3. **Create the Future** — What would make next quarter the best ever, one big win, one habit to start, one to kill, one commitment starting tomorrow

Adapted from Tiago Forte's Annual Review (ARC method). Modified for quarterly cadence by Strategy Sprints.

## What's in the box

- **Act 1 — Appreciate the Quarter** (8 questions) — Gratitude, wins, lessons, risks, joy, speed, surprise, one word
- **Act 2 — Read the Present** (5 questions) — Pain, desire, bottleneck, tolerations, postponed decisions
- **Act 3 — Create the Future** (7 questions) — Vision, biggest win, habits, learning, joy, theme, commitment

## How to use it

## As a Claude Code slash command

```
/quarterly
```

That's it. The interview starts.

## Installation

**Global (available in every Claude Code session):**

```bash
git clone https://github.com/SimonTheSalesBooster/quarterly.git
cp quarterly/.claude/commands/quarterly.md ~/.claude/commands/
```

**Per-project (available when working in that directory):**

```bash
cd your-project
git clone https://github.com/SimonTheSalesBooster/quarterly.git .claude/skills/quarterly
```

## As a system prompt

Paste `quarterly-review.md` into any frontier model conversation. Then say: "Interview me."

## Standalone

Copy the 20 questions and ask them to yourself, a coach, a co-founder, or a friend. The questions work with or without AI.
